# 📊 Sales Data Analysis & Business Insights
**Task 1 - Mains Craft Technologies Virtual Internship**
**Author:** Varun

## 📝 Project Objective
The goal of this project is to perform an end-to-end data analysis on a retail dataset to uncover actionable business insights. This project evaluates regional sales performance, category profitability, the impact of discounting strategies, and seasonal sales trends to provide data-driven recommendations for management.

## 🛠️ Tools & Technologies Used
* **Data Cleaning & EDA:** Microsoft Excel (Pivot Tables, Data Validation)
* **Database & Extraction:** SQL (MySQL / PostgreSQL)
* **Data Visualization:** Power BI / Tableau
* **Reporting:** PDF Business Insights Report

## 📂 Repository Structure
* `/data` - Contains the cleaned dataset (`cleaned_sales_data.xlsx`).
* `/excel_analysis` - Contains the exploratory data analysis and Pivot Tables.
* `/sql_scripts` - Contains the table schema (`01_schema_setup.sql`) and analysis queries (`02_analysis_queries.sql`).
* `/dashboard` - Contains the interactive dashboard file and a high-resolution screenshot.
* `/reports` - Contains the final Business Insights PDF report.

## 💡 Key Insights Discovered
1. **Regional Performance:** The West region generates the highest sales ($725K), while the South region significantly underperforms ($391K).
2. **Category Profitability:** Technology is the primary profit driver ($145K profit). Furniture generates high revenue ($741K) but operates at near-zero profitability ($18K profit) due to heavy discounting.
3. **Discount Impact:** Discounts exceeding 30% severely cannibalize margins, with 50% discounts resulting in an average loss of $310 per transaction.
4. **Seasonal Trends:** The fourth quarter (specifically November and December) drives the vast majority of yearly revenue, requiring scaled supply chain preparation.
